## Installation

1. Go to terminal and clone this repository:
```
$ git clone https://github.com/Ngann/galactic-calculator.git
```
2. Change into the project directory and install npm
```
$ npm install
```
3. Build the app:
```
$ npm run build
```
4. To run the program:
```
$ npm start
```
6. To test the code:
```
$ npm test
```
